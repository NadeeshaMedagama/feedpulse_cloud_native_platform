# web

Frontend assets for FeedPulse_Cloud_Native_Platform.

- `index.html` - public feedback submission page
- `admin.html` - admin dashboard page
- `styles.css` - shared theme and layout styles
- `app.js` - public form interactions
- `admin.js` - admin login, filters, pagination, actions

UI features:
- Professional top navigation and page layout
- Dark theme toggle with icon
- Sentiment badges and stats cards
